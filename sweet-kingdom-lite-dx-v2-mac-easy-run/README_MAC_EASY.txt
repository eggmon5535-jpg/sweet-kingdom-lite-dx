# Sweet Kingdom Lite DX v2 (Mac Easy Run)

## 가장 쉬운 실행 방법
1. 이 폴더를 Finder에서 연다
2. **start.command** 를 더블클릭
3. 처음 실행 시 macOS가 막으면:
   - 우클릭 → 열기
4. 터미널 창이 열리면 자동으로 설치/실행됨
5. 브라우저에서 아래 주소 열기
   - http://localhost:5173

## 종료 방법
- 터미널 창에서 `control + C`

## 안 될 때
- Node.js 먼저 설치:
  - https://nodejs.org/
